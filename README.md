This is my blog, it is powered by [VuePress 2](https://v2.vuepress.vuejs.org/) and themed by [Gungnir](https://github.com/Renovamen/vuepress-theme-gungnir). &nbsp;

## Usage

Clone the repo and install dependencies:

```bash
yarn
```

Start developing the site:

```bash
yarn dev
```

Build for production with minification:

```bash
yarn build
```
